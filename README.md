# Gender Detection Using Machine Learning

## Project Overview

This project predicts the gender of a person based on their name using Machine Learning.

The system is trained on a dataset of Indian names and classifies the input as Male or Female through a Flask web application.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Flask
* HTML
* CSS

## Features

* Name-based gender prediction
* Machine Learning classification model
* Web-based interface
* Confidence percentage display
* Flask deployment
